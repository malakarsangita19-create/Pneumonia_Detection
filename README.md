# Pneumonia Detection using Deep Learning

## Overview
This project detects Pneumonia from Chest X-ray images using Artificial Intelligence and Deep Learning.

## Features
- Chest X-ray image upload
- Pneumonia prediction
- Deep Learning model
- Image preprocessing
- User-friendly interface

## Technologies Used
- Python
- TensorFlow
- OpenCV
- Streamlit
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

Run the project:

```bash
streamlit run app.py
```

## Project Structure

- app.py
- requirements.txt
- logo1.jpeg
- sample xray images
